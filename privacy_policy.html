Language Learning with Netflix Chrome Extension - Privacy Policy

On installation of the extension, and every time the user visits netflix.com, we send information about the user and usage to our server. This information includes:

  • An anonymous user ID, provided by the Chrome browser.
  • The email address associated with the account the user is logged into the Chrome browser with. We do this using an API provided by the Chrome browser. This is used occasionally to contact the user in the event of technical issues. For example: “the extension is currently facing technical issues, but we expect it to be fixed within 48hrs.”
  • Information about whether a referer referered the user to our homepage, languagelearningwithnetflix.com. This information is stored in a cookie by our website, and retrieved by our extension. This is used to aid and direct our marketing effort.
  • The user’s Netflix country and Netflix user language. This is collected when the user watches a video on Netflix.com. This is used to aid and direct our marketing and development effort.
  • The version number of the installed extension.
  • The total number of subtitles the user watched while using the extension. This lets us know how much users are using our extension.
  • The user’s browser version and operating system. This is used to aid our development and debugging effort.

We do not share any of this information with third parties.

When the extension has detected a critical error has occurred, the extension sends a message to our server. The extension also notifies the user that a message has been sent to our server. The message contains:

  • A short description of the error.
  • An anonymous user ID, provided by the Chrome browser.
  • The user’s Netflix country and Netflix user language.
  • The version number of the installed extension.
  • The user’s browser version and operating system.

This information assists us with our debugging effort (fixing our code). We do not share any of this information with third parties.

All data is sent over secure connection, using HTTPS.

No further information about the user or their activities is collected, stored, or transmitted, other than that described above.
